# Payment Getway service

**Real-Time Payment Processing Engine Service**

## Overview:
A high-throughput payment gateway that processes NEFT/RTGS/UPI-style transfers with idempotency, retry logic, and real-time status tracking.

## Core Features
```
✅ Initiate payment (NEFT / RTGS / UPI).
✅ Check payment status in real-time.
✅ Handle duplicate payment requests (idempotency).
✅ Retry failed payments automatically.
✅ Notify sender/receiver on status change.
✅ Reconcile payments at end of day.
✅ Handle failed/stuck payments.
```
